# Feature_extraction
An implementation of the use of deep learning models to learn and visualize representations of image data

This pipeline works for an unlabelled dataset of images and include important preprocessing techniques that include resizing central objects using open cv as well as feature space explorations
using umbedding algorithms such as UMAP, tSNE and PMAP.

The best plase to start is with the explore notebook which goes ove the starting processes of taking a pre-trained CNN and using it to extract features
from a a folder of images. The features can then be visualized on a 2D plane
